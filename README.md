## 📦 仓库性质

⚠️ **本仓库是 Hexo 已构建的静态产物**，不是 Hexo 源仓库。

- ✅ 包含完整的 HTML / CSS / JS，可直接被 GitHub Pages 部署
- ❌ 不包含 `_config.yml`、`source/_posts/`、`themes/` 等源文件
- 🔧 想改主题/样式 → 直接编辑 HTML；想加新文章 → 复制现有 HTML 骨架再改正文

详见博文 [《通过 Agent 维护本博客》](https://blog.rwchen.top/2026/06/22/agent-maintain-blog/)。

## 🤖 Agent 维护说明

本博客从 2026 年 6 月开始，由 AI Agent (OpenClaw) 协作维护：

- 修改 hostname、permalink 等全局配置
